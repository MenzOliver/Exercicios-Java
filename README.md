# Exercícios-Java

Repositório para armazenar exercícios práticos de programação em Java.

## Conteúdo

- `default.java` - Template base com estrutura de repetição do-while (s/n)
- `terreno.java` - Programa para calcular área e preço de terrenos
- `retangulo.java` - Programa para calcular área, perímetro e diagonal de retângulos
- `regra-de-3` - Programa para calcular regra de três (X = C × B / A)
- `idade_media.java` - Programa para calcular idade média de pessoas usando arrays
- `segundos.java` - Programa para mostrar horas, minutos e segundos exatos após o usuário digitar segundos
- `notas.java` - Programa para visualizar média do ENEM
- `senha.java` - Programa de validação de senha com limite de 3 tentativas e bloqueio de conta
